My journey into tech has not been linear, but one with hills and valleys and unexpected detours. My past work experience has included a bundle of different industries like healthcare and insurance, each rewarding in their own way but something was always missing… I needed a challenge that would also engage my creativity. Once I got a glimpse into tech and started bringing ideas to life with code, I knew this is what I wanted to do. While this is a change for me, I believe much of life is determined by how we choose to see things and a growth mindset is everything. Broadening my skill set has been very fulfilling thus far and I’m thrilled to see where this road takes me!

When I’m not immersed in code I enjoy time with family, but you can also catch me with a sketchbook, an audiobook (sometimes both) or on a pickleball court.

projects

super sticky notes

An interactive app built with React that will allow users to create “sticky notes” on their computer to keep track of to-do items. The user can add, write, edit and remove notes as needed. The app was built in CodeSandbox using ES6, JSX, managing data flow and manipulating the DOM.

github repo gallery

This project pulls data from my GitHub portfolio to create a gallery of repos. Visitors to this site can search by name and click on repos to see more details. Built using Vanilla JS, GitHub APIs and manipulating the DOM.

guess the word

Players guess the word by entering one letter at a time. If the player guesses all the letters correctly before they use up their allotted guesses, they win! Built using Vanilla JavaScript and manipulating the DOM

unplugged

A multi-page responsive website coded with Flexbox. This versatile design has several sections, including a Cards design pattern, that can be repurposed for different sites. Built with HTML5, CSS3, Flexbox, Media Queries, and Responsive Typography.

rogue pickings

A simple homepage perfect for a pop-up restaurant or food truck. Originally a static website, I converted it into a responsive design using Flexbox. This site was built using HTML5, CSS3, Flexbox, Media Queries and Responsive Typography.
